# Daily Progress Report Form

This repository contains a simple, ready-to-use HTML form for daily work progress reports, supporting file uploads and integration with Formspree.

## Usage

1. Replace the Formspree URL in `index.html` with your own form endpoint from https://formspree.io/ after creating a form project.
2. Style and behavior are provided by `styles.css` and `script.js`.
3. Deploy the repo to your preferred static hosting (GitHub Pages, Netlify, Vercel, etc.).
